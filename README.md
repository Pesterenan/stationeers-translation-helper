# Stationeers Translation Helper

A simple web application designed to make translating Stationeers localization files easier and less error-prone.

Stationeers uses large XML files for localization, which can be tedious and risky to edit manually. This tool provides a visual, card-based interface to translate entries incrementally and safely.

---

## Goal

- Simplify the translation workflow for Stationeers
- Avoid manual XML editing
- Allow incremental translation with progress saving
- Export ready-to-use XML files for the game

## Tech Stack

- **React**
- **TypeScript**
- **Vite**
- **Browser-only (no backend)**
- **DOMParser / XMLSerializer**

---

## Getting Started

### Requirements
- Node.js >= 18
- npm

### Install & Run

```bash
npm install
npm run dev
```
